[ 페이지 설계 & 기능 ]

1. 메인 페이지
    1) 은행별 실시간 금리 확인(은행연합회 소비자포털)
    2) 분양 정보 및 청약 정보 제공(한국부동산원 청약홈에서 OpenAPI제공)
    3) 추천 금융 상품 및 서비스
    4) 추천수 높은 게시글 리스트 
    5) 최근 인기 있는 목표 리스트
    6) 금융 관련 뉴스 및 트렌드 공유
    7) 사용자에게 힘을 주는 오늘의 명언 제공

2. 인생 시뮬레이터 입력 페이지
    1) 사용자 정보 입력 (나이, 직업, 소득, 관심 분야 등)
    2) 목표 설정 (주택 구매, 투자, 자녀 교육, 노후 준비, 여행, 자동차 구매 등)    

3. 전략 플랜 추천
    1) 사용자 목표에 따른 재무 계획 솔루션 제공
    2) 목표 달성을 위한 단계별 전략 및 금융 상품 추천
    3) 목표 달성을 위한 예상 시나리오 제공
    4) 사용자의 월 소득, 자산, 부채 소비 패턴 분석 후 사용자의 성향에 따라 최적 자금 운용 플랜 또는 최대 운용 가능 자금 플랜등 선택지 추천

4. 통계 및 비교 페이지
    1) 사용자와 같은 연령대 비교 차트
    2) 사용자와 같은 직업군 비교 차트
    3) 사용자와 같은 소득 수준 비교 차트
    4) 사용자와 같은 관심 분야 비교 차트

5. 커뮤니티 & Q&A 게시판
    1) 질문 게시판
        - 사용자 질문 및 답변 게시판
    2) 유저 솔루션 공유 게시판
        - 유저가 시뮬레이터를 통해 얻은 재무 계획 솔루션 & 결과 공유
        - 추천수 높은 글은 홈에 베스트 솔루션으로 노출
        - 유저 솔루션에 대한 좋아요/댓글 기능, 포인트 지급 시스템        
    3) 정보 공유 게시판
        - 금융 상품, 서비스, 투자 정보 공유

6. 순위 페이지
    1) 포인트, 좋아요 수, 기여도 기반 랭킹
    2) 개인 프로필에 목표 히스토리/성취율 표시

7. 로그인/회원가입 
    1) 로그인 기능
    2) 회원가입 기능
        - 사용자 정보 입력 (이름, 아이디, 비밀번호, 이메일, 나이, 직업, 소득, 관심 분야, 거주지 등)
    3) 아이디, 비밀번호 찾기 기능
    
8. 마이페이지 관리 
    1) 회원 정보 수정 기능
    2) 비밀번호 변경 기능
    3) 회원 탈퇴 기능
    4) 목표 설정 및 수정 기능
    5) 목표 달성 현황 확인 기능
    6) 포인트(10번 포인트 페이지 이동) 및 기여도 확인 기능
    7) 사용자 맞춤형 목표 추천 기능
    8) 사용자 맞춤형 금융 상품 추천 기능
    9) 채팅 기록 확인 기능(11번 채팅 페이지 이동)

9. 관리자 페이지
    1) 사용자 관리 기능
        - 사용자 정보 조회, 수정, 삭제
        - 사용자 활동 로그 확인
    2) 게시판 관리 기능
        - 게시글 삭제 기능
        - 댓글 관리 기능
    3) 포인트 및 기여도 관리 기능
        - 포인트 지급/차감 기능
    4) 오픈 채팅방 관리 기능
        - 오픈 채팅방 관리(참여자 관리, 채팅방 삭제 등)

10. 포인트 페이지
    1) 포인트 사용 내역 확인 기능
    2) 포인트 적립 내역 확인 기능
    3) 포인트 사용 가능 서비스 리스트

11. 채팅 페이지(포인트 사용)
    1) 사용자 프로필 클릭 시 1대1 채팅 요청
        - 채팅 요청 시 상대방에게 알림 전송
        - 채팅 요청 수락/거절 기능
    2) 채팅 기록 확인 기능
    3) 채팅 알림 기능
    4) 채팅 차단 기능
    5) 채팅 신고 기능
    6) 채팅방 나가기 기능
    7) 오픈 채팅방 생성 기능
    8) 오픈 채팅방 참여 기능
    9) 오픈 채팅방 목록 확인 기능
    10) 오픈 채팅방 검색 기능
    11) 오픈 채팅방 신고 기능
    12) 오픈 채팅방 나가기 기능



[ 추가 기능 ]

1. 챗봇 기능
    1) 사용자 질문에 대한 자동 응답 기능
    2) 금융 상품 및 서비스 추천 기능
    3) 목표 설정 및 달성 관련 조언 제공
    4) 커뮤니티 게시글 추천 기능

2. 포인트 제공 리스트
    1) 게시글 작성, 댓글 작성에 따라 포인트 지급
    2) 목표 달성 시 포인트 지급
    3) 커뮤니티 활동(좋아요), 추천수에 따른 포인트 지급
    4) 상위 랭킹에 따른 포인트 지급

3. 포인트 사용 리스트
    1) 프리미엄 서비스 이용(고급 통계, 전문가 매칭 이용권 등)
    2) 채팅 기능 이용(1대1 채팅, 오픈 채팅방 참여 등)
    3) 커뮤니티 게시글 상위 노출
    4) 특별 이벤트 참여권 제공
    5) 룰렛, 경품 추첨 참여권 제공(포인트를 많이 쓸수록 확률 증가)
